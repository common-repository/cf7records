Never miss an enquiry again! Save your Contact Form 7 records.

==  Description==


Easy install, no configuration necessary. Once activated all contact form 7 submitted records will be saved, so you can view them in wp-admin contact form submitted records.

Each record submission is stored in the database so they can be easily managed using the default WordPress interface. 
You can filter records by searching for keywords, selecting individual contact forms. 
To show the posted values in the listing table just filter the record submissions by a form.

Files are stored in the /wp-content/uploads directory and can be previewed or downloaded from the single record submission page.

== Installation ==

1. Upload `cf7record` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

You will find 'CF7 Records' menu in your WordPress admin panel under Contact.
